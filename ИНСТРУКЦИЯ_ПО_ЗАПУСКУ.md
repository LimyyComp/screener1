# Инструкция по запуску приложения

## Шаг 1: Установка Node.js

Если у вас не установлен Node.js, скачайте и установите его:
- Перейдите на https://nodejs.org/
- Скачайте LTS версию (рекомендуется)
- Установите Node.js
- Перезапустите терминал/командную строку

Проверьте установку:
```bash
node --version
npm --version
```

## Шаг 2: Открытие проекта

Откройте терминал (PowerShell или Command Prompt) и перейдите в папку проекта:

**Вариант 1: Через проводник**
1. Откройте папку проекта в проводнике
2. В адресной строке введите `cmd` и нажмите Enter
3. Или нажмите Shift + ПКМ в папке и выберите "Открыть окно PowerShell здесь"

**Вариант 2: Через терминал**
```bash
cd "C:\Users\User\Desktop\апка"
```

## Шаг 3: Установка зависимостей

Выполните команду:
```bash
npm install
```

Это установит все необходимые библиотеки (React, TypeScript, Tailwind CSS, Zustand, TradingView Charts и т.д.)

## Шаг 4: Запуск приложения

После установки зависимостей выполните:
```bash
npm start
```

Приложение автоматически:
- Скомпилируется
- Откроется в браузере на адресе http://localhost:3000
- Начнет получать данные от Binance в реальном времени

## Что вы увидите:

1. **Таблица с криптовалютными парами** - обновляется в реальном времени
2. **Панель управления сверху** с:
   - Переключателем Spot/Futures
   - Кнопками таймфреймов (1m, 5m, 15m, 1h, 4h, 1d)
   - Сортировкой
   - Поиском
   - Переключателем темы

3. **Клик по паре** - откроет детальный график

## Остановка приложения

Нажмите `Ctrl + C` в терминале, чтобы остановить сервер разработки.

## Возможные проблемы:

### Ошибка "npm не является командой"
- Убедитесь, что Node.js установлен
- Перезапустите терминал
- Проверьте, что Node.js добавлен в PATH

### Ошибка порта занят
- Закройте другие приложения на порту 3000
- Или используйте другой порт: `PORT=3001 npm start`

### Проблемы с кириллицей в пути
- Попробуйте переместить проект в папку с английским названием
- Или используйте короткий путь через `subst`
