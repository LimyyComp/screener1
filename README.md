# Crypto Screener

Полнофункциональное веб-приложение для крипто-скрининга с данными в реальном времени от Binance.

## Технологии

- **React** + **TypeScript** - основной фреймворк
- **Tailwind CSS** - стилизация
- **Zustand** - управление состоянием
- **TradingView Lightweight Charts** - графики
- **Binance WebSocket API** - данные в реальном времени

## Функционал

- ✅ Отображение криптовалютных пар в реальном времени
- ✅ Обновление цен, объемов и изменений без задержек
- ✅ Мини-графики (спарклайны) для каждой пары
- ✅ Детальный свечной график при клике на пару
- ✅ Фильтрация и сортировка по различным параметрам
- ✅ Переключение между Spot и Futures рынками
- ✅ Выбор таймфрейма (1m, 5m, 15m, 1h, 4h, 1d)
- ✅ Поиск по символам
- ✅ Темная/светлая тема

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Приложение откроется на [http://localhost:3000](http://localhost:3000)

## Сборка для продакшена

```bash
npm run build
```
