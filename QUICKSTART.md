# Быстрый старт

## Установка зависимостей

```bash
npm install
```

## Запуск приложения

```bash
npm start
```

Приложение автоматически откроется в браузере на `http://localhost:3000`

## Основные возможности

1. **Реальное время**: Данные обновляются автоматически через WebSocket
2. **Spot/Futures**: Переключение между спотовым и фьючерсным рынками
3. **Таймфреймы**: Выбор интервала (1m, 5m, 15m, 1h, 4h, 1d)
4. **Сортировка**: По объему, изменению цены, цене или символу
5. **Поиск**: Фильтрация пар по символу
6. **Графики**: Клик по паре открывает детальный свечной график
7. **Темы**: Переключение между светлой и темной темой

## Структура данных

Приложение использует Binance API:
- **Spot**: `wss://stream.binance.com:9443/ws/`
- **Futures**: `wss://fstream.binance.com/ws/`

## Производительность

- Оптимизировано для работы с 100+ парами одновременно
- Обновления данных без перезагрузки страницы
- Плавные анимации при изменении цен
